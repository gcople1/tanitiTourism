# Island of Taniti Tourism Website

## Project Overview
This project is a comprehensive tourism website designed for the island of Taniti. It serves as a one-stop digital hub for visitors, providing essential information on travel logistics, accommodations, local dining, activities, and island-specific FAQs. [cite_start]The site is structured to be user-friendly with a clear navigation system and interactive widgets to assist in trip planning[cite: 1, 138].

## Site Architecture
[cite_start]The website is organized into several key sections accessible via a primary navigation menu [cite: 9-15]:
* [cite_start]**Home/Landing Page:** A welcoming entry point featuring a high-level overview and quick access to major site categories[cite: 1, 138].
* [cite_start]**Travel:** Information on arriving at Taniti by plane or ship, including booking widgets and details on airport expansions [cite: 17-21, 130].
* [cite_start]**Stay (Accommodations):** Detailed sections for hotels, resorts, and hostels with integrated booking widgets and location maps [cite: 28-31, 131].
* [cite_start]**Eat (Food):** A guide to local dining categorized by location (Taniti City, Merriton Landing) and cuisine type (Local, Pan-Asian, American), as well as grocery information [cite: 52-60, 137].
* [cite_start]**Play (Activities):** A robust catalog of things to do, filterable by activity type (Outdoor, Arts & Culture, Guided Tours) or location [cite: 67-84, 140].
* [cite_start]**Get Around (Transportation):** Logistics for island travel, including public buses, taxis, rideshares, car rentals, and bicycle rentals (with legal helmet requirements) [cite: 89-108, 139].
* **Ask! (FAQ & Contact)[cite_start]:** Essential visitor information and a communication portal for support [cite: 115-121, 141].

## Key Features
* [cite_start]**Interactive Booking Widgets:** Users can book flights, cruises, and various types of lodging directly through the site [cite: 20-21, 31, 40].
* [cite_start]**Dynamic Mapping:** Multiple pages feature draggable and zoomable maps with marked locations to help visitors navigate attractions and services[cite: 23, 61, 85, 102].
* [cite_start]**Categorized Activity Discovery:** Includes a "Rainy Day Fun" section to ensure visitors have options regardless of weather[cite: 78].
* **Essential Visitor Logistics:** The FAQ section provides critical data on:
    * [cite_start]**Currency:** US Dollar is the standard, though Euros and Yen are often accepted[cite: 127].
    * [cite_start]**Power:** 120-volt outlets (US standard)[cite: 119].
    * [cite_start]**Laws & Safety:** Information on drinking age (18), alcohol service hours, and safety tips regarding petty crime[cite: 120, 122, 125].
    * [cite_start]**Language:** Guidance on where English is fluently spoken versus rural areas where it is less common[cite: 123].

## User Experience (UX) Design
* [cite_start]**Navigation:** The site utilizes a consistent navigation bar and a "hamburger menu" for mobile accessibility [cite: 1, 9-15].
* [cite_start]**Site Flow:** The architecture ensures that users can move from high-level information (Landing Page) to specific external booking tasks with minimal clicks [cite: 129-142].
* [cite_start]**Breadcrumbs:** Pages are designed with clear headers to help users identify their current location within the site hierarchy [cite: 24-28, 44-49].
